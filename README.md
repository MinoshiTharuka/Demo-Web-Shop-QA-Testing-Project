# Demo Web Shop QA Testing Project

## Project Overview
The project focuses on performing both manual testing and automation testing for the Demo Web Shop e-commerce application https://demowebshop.tricentis.com/. The testing process included requirement analysis, test planning, test case creation, defect reporting, and Selenium test automation.

---

## Objectives
- Validate core functionalities of the web application
- Identify defects and usability issues
- Perform manual and automated testing
- Improve software quality through structured QA practices

---

## Application Features Tested
- User Registration
- User Login
- Product Search
- Add to Cart
- Shopping Cart Updates
- Checkout Process
- UI/UX Validation
- Error Message Validation
